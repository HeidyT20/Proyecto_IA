# Decisiones del Módulo de Machine Learning

**Algoritmo seleccionado:** Random Forest Regressor.
**Razón:** Es excelente para manejar datos tabulares de ventas y capturar relaciones no lineales entre promociones y días de la semana.

**Características (Features) utilizadas:**
- Tienda (Store)
- Día de la semana (DayOfWeek)
- Fecha (Año, Mes, Día separados)
- Promociones (Promo)
- Feriados (StateHoliday)

**Objetivo:** Predecir la columna `Sales` para optimizar el inventario.
