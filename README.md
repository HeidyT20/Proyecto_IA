\## 📊 Gestión de Datos (Semana 2)



\*\*Datos:\*\* El dataset utilizado para este proyecto es `train.csv` de Rossmann Store Sales. 



⚠️ \*\*Nota sobre el almacenamiento:\*\* Debido a que el archivo supera el límite de tamaño recomendado para GitHub (36.2 MB), no se encuentra alojado en este repositorio. 



\*\*Instrucciones para ejecución:\*\*

1\. Descargar el dataset desde \[Kaggle: Rossmann Store Sales](https://www.kaggle.com/c/rossmann-store-sales/data).

2\. Colocar el archivo `train.csv` en la ruta: `src/data/`.

3\. Ejecutar el script `load\_data.py` para verificar la carga.
